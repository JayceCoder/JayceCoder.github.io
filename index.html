使用hexo搭建个人博客

  1. 安装 git安装完之后使用cmd命令查看是否安装成功 （git –version）
  2. 安装 node.js安装完之后使用cmd命令查看是否安装成功 （node –version）
  3. 安装hexo
    在桌面任意位置单击右键，选择Git Base
    输入命令
    npm install -g hexo
    注:-g表示全局安装
  4. 在任意位置(你需要存放博客的本地地址)先创建一个hexo文件夹<
